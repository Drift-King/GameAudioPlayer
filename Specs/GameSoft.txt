=============================================================================
		     Game Software Links/Mails Compilation	   15-04-2001
=============================================================================
		      Compiled by Valery V. Anisimovsky (son_of_the_north@mail.ru)

This compilation contains links to game-related utilities (and other stuff)
and mails of their authors. Utilities and stuff include: extractors/rippers
for game files, players/converters for game audio/video files, game file
formats specifications, game audio/video files. Info includes: URLs, e-mail
addresses, ICQ numbers.

The distribution of this compilation is thankfully appreciated as long as
it's distributed without any destructive changes which haven't been discussed
with the author. Some links and addresses may appear to be valid no more (or
temporarily unavailable). This does not mean that the correspondent link/address
should be removed from the compilation -- it may start to work again in the
future! Any additions are appreciated, but note that this compilation is aimed
mostly at game audio/video conversion/playback, game resources extraction
and game audio/video/resource file specifications. The most consistent way
to add new info to this compilation (at least in my opinion) is to place the
info into the appropriate section and add the e-mail address of the adder to
the list of contributors at the end of compilation. Sending new info to the
author also looks like a good idea.
