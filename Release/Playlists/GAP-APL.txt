=============================================================================
			ANX Playlists for CD games		   07-03-2002
=============================================================================

Premade APLs (GAP playlists) for CD games may be used on systems with
different CD drive letters, as long as GAP is properly configured to
save/load such APLs and APLs are made for game data residing on the game CD.

You are encouraged to add correct (usable) APLs for CD games to this package
(and, correspondingly, add your name and contact info along with your APLs
descriptions to this text file). But really useful are the APLs with either
of two features (better with both):
1) containing audio file names: songnames for music files, game context based
descriptions of sound/speech files,
2) containing audio files which were found as a result of scanning of a large
resource file or a massive bunch of files on game CD.
Such playlists are quite difficult to create, so it's always good to distribute
them. Especially useful are APLs with songnames, as this information is rarely
stored in game audio/resource files, thus, it often cannot be obtained by GAP
itself.

GAP-APL.ZIP package includes ANX Playlists for the games:
